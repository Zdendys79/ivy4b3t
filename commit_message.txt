feat(debug): Add diagnostic incident reporting to findByText

- Implemented a diagnostic mechanism in the `findByText` function in `iv_fb_support.js`.
- When the function fails to find a requested element, it now automatically captures the full page HTML, a screenshot, and other context.
- This data is then saved as a new record in the `debug_incidents` database table.
- This replaces the previous, less practical approach of saving local files and provides a centralized way to analyze UI detection failures.
