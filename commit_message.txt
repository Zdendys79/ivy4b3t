fix(ConsoleLogger): Implement robust recursive sanitization

- Replaced the simple string replacement with a recursive JSON.stringify replacer.
- This new implementation traverses nested objects and arrays, replacing all occurrences of double quotes and backslashes to ensure that the resulting string is always safe for SQL insertion.
- This definitively resolves the recurring `logs.insertConsoleLogBatch` failures, which were the primary source of application instability and debugger spam.
