fix(worker): Oprava parsování Git informací

Tato oprava řeší pád aplikace `Unexpected end of JSON input`, který nastával při pokusu o zjištění aktuální Git větve.

- **feat(git): Přidána funkce `get_git_info_json` do `git-common.sh`**
  - Původní funkce vracela prostý text, který nebylo možné parsovat jako JSON.
  - Nová funkce vrací informace o repozitáři ve správném JSON formátu.

- **fix(support): Aktualizována funkce `getGitInfo` v `iv_support.js`**
  - Funkce nyní volá novou `get_git_info_json` a obsahuje robustnější ošetření chyb.
  - V případě selhání parsování nebo chyby skriptu nyní vrátí výchozí objekt, aby se zabránilo pádu celé aplikace.
