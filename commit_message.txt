fix(debugger,worker): Finální opravy pro exit code a blokování účtů

Dokončuje řešení problémů s interactive debuggerem a account blockingem.

- **fix(debugger): 'q' key nyní používá exit code 99**
  - Změna z SIGINT na process.exit(99) pro správnou detekci v start.sh
  - Umožní start.sh správně detekovat kontrolované ukončení
  - Zabrání nekonečným restartům po stisknutí 'q'
- **fix(worker): Zajištění ukončení tick() po account blocking**
  - Explicitní komentář u return statement pro clarity
  - Garantuje že po zablokování účtu se tick() ukončí
  - Předchází pokračování v worker loop s problematickým účtem
