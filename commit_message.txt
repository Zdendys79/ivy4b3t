fix(posting): Refactor send button logic and improve error logging

- Completely refactored the `clickSendButton` method in `iv_fb.class.js` to use the robust `findByText` helper function instead of a fragile `page.$$('span')` loop. This ensures it reliably finds the send button across different element types.
- Improved error logging in `iv_support.js` and `iv_actions.js` to provide more specific messages, distinguishing between a failure to get a message from UTIO and a failure to post it on Facebook.
- This fixes a critical bug where the bot would fail to send a post and incorrectly report an issue with UTIO.
