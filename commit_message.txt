fix(fb): Correct element selection in clickSendButton

- Fixed a critical `TypeError: spans is not iterable` in the `clickSendButton` method.
- The error was caused by incorrectly using `page.$('span')` (which returns a single element) instead of `page.$$('span')` (which returns an array of elements).
- This correction ensures that the code can properly iterate over all found `<span>` elements to locate the correct send button, resolving the crash during the posting process.
