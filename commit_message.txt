fix(Core): Final stabilization and bug fixes

- fix(ConsoleLogger): Added a `[LOGGER-FAIL]` prefix to error messages originating from the logger itself, making them clearly distinguishable from application errors.
- fix(Debugger): Resolved a memory leak by properly cleaning up `stdin` listeners after user input. This prevents the `MaxListenersExceededWarning` and ensures the debugger is stable.
- fix(Support): Corrected a `wait is not defined` runtime error by adding the proper `iv_wait.js` module import to `iv_support.js`. This resolves the crash that occurred after successfully fetching a message from UTIO.
