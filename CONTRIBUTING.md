# Hlavní cíl projektu IVY:
- Vyvíjíme a spravujeme sadu Puppeteer botů, kteří ovládají účty virtuálních uživatelů Facebooku. Tito boti se chovají jako reální uživatelé, vykonávají akce jako přidávání přátel, lajkování příspěvků, odesílání zpráv, komentování atd.
- Hlavním účelem je přirozené chování a interakce s cílovými skupinami a uživateli.
